The employed model is an combination of MultiBOSS transform and an SVM classifier. The MultiBOSS, which stands for Multivariate Bag of Symbolic Fourier Approximation Symbols, is a dictionary based learning framework based on combining the SFA and bag-of-words concepts. The univariate BOSS transform works by running a sliding window of fixed length along each series, applying normalization to focus on the shape fo the signal, and applying Discrete Fourier Transform (DFT). The DC component is removed and only the low-frequency components are kept to focus on the global shape. Using Multiple Component Binning (MCB), the resulting DFT is quantized into discrete letters to form a combination of discrete symbol (MCB). Athen shortening and discretising each window to form a combination of discrete symbols (word). It counts the frequency of each word to form a histogram of word counts.
